# WSL Extension

## Overview

WSL Extension is a lightweight tool that empowers users to capture valuable content directly from the web. Whether you need to save a short snippet of text or extract the main article content from a page, WSL Extension makes it quick and easy with just a click or keyboard shortcut.

## Use Case

Every day, we come across web pages filled with information worth saving for later referenceâ€”be it for research, writing, or personal interest. WSL Extension allows you to instantly capture a selected snippet or the entire readable content of a page. By streamlining the capture process with keyboard shortcuts and background functionality, it helps you keep track of useful content without interrupting your browsing experience.

## Features

- **Capture Snippets:** Quickly save selected text from any webpage.
- **Full Page Capture:** Utilize Mozilla's Readability to extract and save the primary content of a webpage.
- **Keyboard Shortcuts:** Easily toggle the extension (e.g., Ctrl+B) and capture full page content (e.g., Alt+B) with simple shortcuts.
- **Security Focused:** Implements a strict Content Security Policy (CSP) and sanitizes inputs to protect against potential threats.
- **Lightweight & Fast:** Designed to be unobtrusive and efficient, ensuring a smooth browsing experience.

## Security

WSL Extension prioritizes your safety:
- **Content Security Policy (CSP):** A robust CSP limits the sources for scripts and other assets, minimizing exposure to malicious content.
- **Input Sanitization:** User-selected content is sanitized to avoid the risk of storing or executing harmful scripts.

## Installation

1. **Clone or Download:** Obtain the extension's source code from the repository.
2. **Load the Extension:** 
   - Open your browser's extension management page (e.g., `chrome://extensions` or `edge://extensions`).
   - Enable Developer Mode.
   - Click on "Load unpacked" and select the extension folder.
3. **Start Capturing:** Once loaded, click the extension icon or use the specified keyboard shortcuts to capture content.

## Usage

- **Toggle Capture Mode:** Click the extension icon or press **Ctrl+B** to switch the extension on or off.
- **Capture Snippets:** With the extension enabled, simply highlight text on any page to automatically capture it.
- **Capture Full Page Content:** Press **Alt+B** to extract and save the main content of the current page using Mozilla Readability.
